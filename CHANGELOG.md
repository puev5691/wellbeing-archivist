# Журнал изменений

В этом файле фиксируются заметные изменения репозитория.

## 0.2.0 - 2026-04-09

Второе практическое расширение MVP, сосредоточенное на поддержке маршрутизации файлов внутри пакетов чатов.

Добавлено:

- разбор `.wb-copy-map.tsv`
- CLI-команда `show-copy-map`
- генерация черновика карты раскладки
- мягкая классификация файлов пакета
- проверка покрытия карты раскладки
- рекомендации по пополнению `.wb-copy-map.tsv`

Улучшено:

- работа с пакетами чатов стала ближе к реальному процессу раскладки
- диагностика оформленности пакета стала практичнее
- CLI-интерфейс отражает более широкий рабочий цикл

Отдельно зафиксированы операционные уроки:

- команды для Termux следует выдавать в `bash`-блоках
- режим выдачи команд и режим выдачи содержимого файлов нужно различать
- новые CLI-команды следует сначала проверять через help, прежде чем считать их доступными

## 0.1.0 - 2026-04-09

Первая опубликованная рабочая версия MVP.

Добавлено:

- структура репозитория
- SQLite-индекс
- индексация и переиндексация
- поиск по имени файла
- поиск по тексту
- распознавание пакета чата
- обкатка на реальном пакете чата
- первая публикация репозитория в GitHub
